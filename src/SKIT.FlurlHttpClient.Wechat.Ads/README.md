基于 `Flurl.Http` 的微信广告平台 API 版客户端。

---

### 【功能特性】

-   基于微信广告平台 API 封装。
-   请求时自动生成服务商身份令牌，无需开发者手动干预。
-   提供了微信广告平台 API 所需的 MD5 等算法工具类。
